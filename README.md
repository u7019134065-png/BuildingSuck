# XDustry

XDustry is a mod for Mindustry focused on adding experimental gameplay, unusual mechanics, new blocks, production chains, liquids, and technology ideas that expand the original sandbox experience.

The goal of the mod is not only to add stronger content, but also to create new ways to build factories, defend bases, and automate complex systems. XDustry is designed for players who enjoy testing strange combinations, improving efficiency, and experimenting with large industrial networks.

## Planned Content

- New production blocks
- Experimental factories
- Advanced conveyors and transport systems
- Custom liquids and materials
- Powerful defensive turrets
- Additional power generation methods
- Sandbox testing structures
- Technology progression expansion
- Visual effects and unique mechanics

## About The Mod

XDustry is being developed as a long-term project. The mod may include unstable or experimental features while new systems are being tested and improved.

The project is inspired by large automation systems, industrial design, and creative sandbox gameplay. Many ideas are focused on giving players more freedom when designing factories and defense systems.

## Compatibility

Designed for the latest versions of Mindustry.
